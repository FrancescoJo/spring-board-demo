==== Error situations
|===
| HTTP Code | Cause | Reason

| `+400+`
| `+WrongPasswordException+`
| Wrong `oldPassword` is given from a client.

| `+400+`
| `+DuplicatedPasswordException+`
| There are no changes between old and new password.
